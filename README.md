# Gonfig

A simple module to load configurations file to a struct.


## Files supported

| Extension | is supported?     |
|:---------:|:-----------------:|
| .json     |:heavy_check_mark: |
| .xml      |:x:                |
| .yaml     |:x:                |
| .toml     |:x:                |
